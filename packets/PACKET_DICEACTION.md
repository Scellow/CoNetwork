**DiceAction Types: ** These enumeration type values are used to request or control various dice actions in the DiceKing mini game. Contributed by [[http://chained2pvp.com/user/255-pro4never/|Pro4Never]].  
   
        ChipIn = 0,
        ChipInConfirm = 1,
        CancelChip = 2,
        CancelChipConfirm = 3,
        BeginChip = 4,
        EndChip = 5,
        Dice = 6
